How to Use


Strict Mode: Enabling this ensures that pronouns like “it” do not stand for non-names like “school”. See limitations.


Limitations


  1. Strict vs Non-Strict
    “It was fine.” is *technically* a valid restatement for “How was school today?”, as “it” stands for “school.” However, it is not a true restatement of the question. Checking the “strict” checkmark makes it so that only full restatements such as “School was fine today.” make it past the detector.
  2. Preceding Prepositions
    Questions like "Of what country did he come from?" are reinterpreted as "What country did he come from?", to simplify processing.
  3. Accuracy
    ReQuestion.Io is still being tested, but so far we have around 90-95% Accuracy!
